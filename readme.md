# Yvette's Bridal & Formal

## Objetivo del proyecto

Desarrollar un sitio web estático para la tienda "Yvette's Bridal & Formal", que permita
mostrar su catálogo de vestidos de novia, vestidos de graduación y ropa formal. El sitio
incluye páginas de inicio, tienda, carrito, contacto y panel de administración.

## Flujo de trabajo usado

Se utilizó un flujo basado en **Git Flow simplificado**:

1. La rama `main` contiene el código estable y listo para producción.
2. La rama `dev` se usó para el desarrollo general de nuevas funciones.
3. La rama `design` se usó para cambios de estilo y diseño visual.
4. Los cambios se integraron a `main` mediante **Pull Requests** con revisión de código.
5. Se simuló y resolvió un **conflicto de merge** entre las ramas `main` y `dev`.
