OBJETIVO DEL PROYECTO 

Este proyecto está pensado para mejorar la experiencia del usuario mediante la optimización del diseño de la página, adaptándola para personas con discapacidad visual e incorporando nuevas funcionalidades que faciliten la interacción y navegación. Consiste en desarrollar y mejorar una plataforma web para la venta y exhibición de vestidos de novia, trajes para hombre y otros atuendos formales, como vestidos para invitadas y damas de honor, mediante la implementación de un diseño más atractivo, funcional e inclusivo.

El proyecto busca optimizar la experiencia del usuario incorporando herramientas de accesibilidad, como el aumento del tamaño de letra para personas con discapacidad visual, además de integrar funciones interactivas como “me gusta”, carrito de compras y mejoras en la navegación, facilitando la búsqueda, selección y compra de productos. Su objetivo principal es ofrecer la mejor experiencia posible a los clientes, proporcionando una plataforma más accesible, moderna, intuitiva y funcional.
